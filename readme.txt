Remove query strings from static resources like CSS & JS files.

== Description ==

Query String Remover From Static Resources automatically removes query strings from your static resources like CSS and JavaScript files. It will help to improve your site speed and the overall score in Google PageSpeed, YSlow, Pingdom and GTmetrix when you enable cache. Just install and forget rest , as no configuration needed.

After installing “Query String Remover From Static Resources” just empty your cached resources if you are using a cache plugin already and the magic will happen automatically


Contact Author at http://www.hibiscustechnolab.com for more details.

== Installation ==

1. Upload the `hibiscus-query-string-remover` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Done.
